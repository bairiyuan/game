# 桌宠内置小游戏需求说明

## 一、项目背景说明

本桌宠产品的核心定位并非“游戏工具”，而是长期运行在桌面的陪伴型产品。
 内置小游戏的存在目的，是为用户提供一种**短时间、低负担的情绪调节方式**，而不是引导用户进行长时间沉浸。

结合已有调研结论与同类产品经验，内置小游戏更适合作为一种“随时可用、随时可停”的轻量功能：
 用户对内置游戏的可玩性预期并不高，但希望**画面清晰、规则直观、上手即玩**，能够在碎片时间中获得短暂放松，而不会产生浪费时间或被强迫停留的负面体验。

因此，本方案从一开始就刻意避免复杂系统、长线养成与高挫败设计，而是围绕桌宠本体，设计数个**低学习成本、短局制、正反馈明显**的小体量玩法。

------

## 二、我提出的四个内置小游戏玩法说明

### 重力翻转（参考像素小鸟，为避免玩法和跑酷重复+多次点击操作稍作修改）

该玩法为轻反应类小游戏。角色持续向前移动，有障碍物向角色移动，只有中间可通过，玩家通过鼠标点击控制重力方向翻转，通过障碍物获得积分。
 整个游戏只需要“是否点击”这一种判断，不存在复杂操作，但对节奏与反应有一定要求。

该玩法的优势在于：
 操作成本极低、失败原因清晰、单局时间可控，非常适合桌面环境下的碎片化使用，不会对用户当前的主要任务造成干扰。

------

### 躲避流星

流星会从屏幕四周随机飞向桌宠角色，玩家通过拖动角色进行躲避。
 每成功躲避一次即可获得积分，发生碰撞则结束本局。

该玩法强调即时反馈和直观规则，不依赖复杂数值或成长体系。
 用户在任何时刻都可以开始或结束游戏，既不会产生“必须打完一局”的心理压力，也适合在短暂空闲中反复游玩。

------

### 跳跳乐

跳跳乐为节奏控制类玩法。角色在平台间持续前进，玩家通过点击或少量键盘操作控制跳跃时机，避免坠落。成功跳跃获得积分，跳跃到中间积分加倍，失败则游戏结束。
 玩法成熟、用户认知成本低，同时具备一定操作空间，能够避免“过于简单而无聊”的问题。

该玩法在微信小游戏等平台已有广泛验证，适合在桌宠中作为相对“耐玩度稍高”的选项，但依然保持单局时间短、失败后可快速重来的特性。

------

### 疯狂的鸽子（近期爆火）

在该玩法中，鸽子围绕屏幕中心顺时针或逆时针旋转。
 屏幕中央存在一个持续旋转的圆环，圆环上随机分布水果与炸弹（水果和炸弹可以修改为其他东西）。

玩家需要控制角色的旋转方向与时机，使其吃到水果、避开炸弹。
 每成功吃掉一个水果即可获得积分，触碰炸弹则结束游戏。

该玩法的重点不在于高难度，而在于**节奏感与趣味性**：
 旋转运动与桌面环境高度融合，玩法直观但具有一定变化空间，整体体验偏解压而非竞技，有助于增强桌宠的“活着”的感觉。

------

## 三、需求合理性与客户价值说明

从产品定位上看，本方案中的小游戏并不试图成为独立卖点，而是作为桌宠体验的一部分存在。
 所有玩法都遵循几个明确的底层原则：**单局时间短、操作简单、可随时退出、不制造挫败情绪**。

从用户体验上看，这类小游戏不会与桌宠的主功能形成竞争，而是补足“短时间消遣”的空档，既提供正反馈，又不会让用户产生负罪感或被强制停留的不适。

整体而言，该桌宠小游戏方案更像是一种**情绪缓冲层**：
 它不抢占注意力，但在需要的时候始终可用，这一点与桌宠产品的长期陪伴定位高度一致。
